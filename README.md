1.  Présentation du projet
Security Platform - DevSecOps & Pentesting

Application web permettant d’exécuter des scans de sécurité automatisés
(pentesting et DevSecOps) avec génération de rapports IA.

2.  Installation du projet
git clone <repo>
cd security-app

3.  Installation backend
cd backend
npm install

4.  Installation frontend
cd frontend
npm install
npm start

5.  Lancer Docker (outils)

6.  Lancer backend
cd backend
node server.js

8. IA 
Les résultats sont envoyés au module IA pour analyse et recommandations.# Projet_PFA
